# 🧠 Elite Enemies 精英怪

本仓库为《逃离鸭科夫（Escape From Duckov）》模组 **Elite Enemies** 的开源代码。  
模组为游戏加入了可配置的 **精英敌人系统**，包含词条、行为逻辑、掉落与视觉特效等内容。

项目在开发过程中使用了 **AI** 来美化文档并调优部分代码，本人手动设计与完善了整体设计框架与功能实现。

---

## 📌 项目简介

仓库包含以下核心模块：

- 精英词条（Affix）逻辑与数值系统  
- 战斗行为与特殊效果 
- 独立掉落池与词条专属掉落配置  
- ModSetting 配置项与设置界面  
- 多语言本地化文件
- 调试工具等辅助功能
